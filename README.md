# Ricardo Jorge — CV

Ricardo Jorge's one-page CV, built with Next.js and designed for the web and
clean A4 printing.

## Content

The CV is generated from the Markdown files in `content/`. Update those files
to change the profile, experience, projects, education, or skills without
editing the React components. The supported format is documented in
`content/README.md`.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Use the **Download PDF**
button to open the browser's print dialog.

## Checks

```bash
npm run typecheck
npm run lint
npm run build
```
